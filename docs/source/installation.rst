Installation
============

Installing Kubeflow SDK
-----------------------

The Kubeflow SDK can be installed using pip:

.. code-block:: bash

    pip install -U kubeflow

Requirements
------------

- Python 3.9 or higher
- Kubernetes cluster (for Kubernetes backend)
- Test commit for workflows

Optional Dependencies
---------------------

For development:

.. code-block:: bash

    pip install -U kubeflow[dev]
